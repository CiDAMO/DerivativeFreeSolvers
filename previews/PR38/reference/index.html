DerivativeFreeSolvers.coordinate_search — Method
coordinate_search(nlp)

The coordinate search is a derivative free optimization method. It minimizes a certain function f by walking on a grid and taking a step towards smaller function values. When it can no longer find any smaller values, it reduces the grid by a factor of β, repeating the process.

DerivativeFreeSolvers.mads — Method
mads(nlp)

MADS is implemented following the description of

Mark A. Abramson, Charles Audet, J. E. Dennis Jr., and Sébastien Le Digabel.
OrthoMADS: A Deterministic MADS Instance with Orthogonal Directions.
SIAM Journal on Optimization, 2009, Vol. 20, No. 2, pp. 948-966,

But using random vectors instead of Halton vectors, as suggested by Francisco Sobral.
